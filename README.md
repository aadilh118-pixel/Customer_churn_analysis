# Customer_churn_analysis

An interactive Machine Learning web application that predicts customer churn using the Telco Customer Churn dataset. Built with Streamlit and Scikit-learn, this app provides real-time predictions, visual analytics, and business insights.

##  Features

- Real-time customer churn prediction
- Interactive Streamlit dashboard
- Customer churn probability gauge
- Data insights and visualization
- Feature importance analysis
- Prediction history tracking
- Downloadable prediction reports


## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit Lottie Animations

##  Dataset

Dataset used:
- Telco Customer Churn Dataset

The dataset contains customer information such as:
- Contract type
- Monthly charges
- Internet service
- Payment method
- Tenure
- Churn status

##  Machine Learning Workflow

1. Data Loading & Cleaning
2. Data Preprocessing
3. Label Encoding
4. Feature Scaling
5. Train-Test Split
6. Random Forest Model Training
7. Real-Time Prediction
8. Visualization & Insights

##  Model Used

- Random Forest Classifier

The model predicts whether a customer is likely to churn based on user input.


##  Dashboard Insights

The application provides:
- Churn Distribution Pie Chart
- Contract Type vs Churn Analysis
- Top Feature Importance Visualization
- Customer Prediction Reports


## How to Run the Project

### 1️ Clone the Repository

```bash
git clone https://github.com/yourusername/customer-churn-prediction.git
```

### 2️ Navigate to Project Folder

```bash
cd customer-churn-prediction
```

### 3️ Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4️ Run the Streamlit App

```bash
streamlit run app.py
```
